# 局域网文件管理服务器

管理员上传、管理文件；局域网用户浏览和下载。软件自动更新为可选功能。

## 快速启动

```bash
npm install
node app.js
```

Windows 也可双击批处理：

| 文件 | 说明 |
|------|------|
| `start.bat` | 启动服务 |
| `stop.bat` | 停止服务（默认端口 3000） |
| `restart.bat` | 重启服务（新窗口） |

> 批处理内提示为英文，避免 Windows 编码问题。若修改了 `config.json` 中的 `port`，请同步修改 `stop.bat` 里的 `PORT=3000`。

## 停止服务器

**方式一：批处理**

双击 `stop.bat`

**方式二：命令行窗口**

在运行 `node app.js` 的窗口按 `Ctrl + C`，或直接关闭该窗口。

**方式三：手动结束进程**

```powershell
netstat -ano | findstr :3000
Stop-Process -Id <PID> -Force
```

> 默认端口为 `3000`，若修改了 `config.json` 中的 `port`，`stop.bat` 也需同步修改其中的 `PORT` 变量。

## 重复启动说明

服务**不会**自动覆盖旧进程。若端口已被占用，再次启动会报错 `EADDRINUSE`。

请先停止旧服务，再重新启动。

## 使用

| 角色 | 能力 |
|------|------|
| 访客 | 浏览文件夹、下载文件 |
| 管理员 | 上传、新建文件夹、重命名、删除、设为发布版本 |

登录：首页右上角「登录」，或访问 `/login`（默认 admin / 123456）

**操作**：图标/列表切换 · Ctrl/Shift 多选 · 右键菜单 · 拖拽上传（需登录）· 双击打开文件夹/下载文件

## 接口

| 地址 | 说明 |
|------|------|
| `/` | 文件管理器 |
| `/api/entries?folderId=` | 目录内容 |
| `/download/:id` | 下载文件 |
| `/version.json` | 软件更新（可选） |
